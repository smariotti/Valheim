# Dronecam

Mod to implement a drone for a Valehim world.

## Controlling It

```
[DroneCam] Commands:
  /dronecam help
  /dronecam on                               enter free-fly setup
  /dronecam off                              return to normal camera
  /dronecam players                          list visible players
  /dronecam follow <player> [dist] [height]  chase a player
  /dronecam orbit player <n> [r] [spd] [h]   orbit a player
  /dronecam orbit pos [r] [spd] [h]          orbit current look-at position
  /dronecam orbit speed <deg/sec>            change orbit speed live
  /dronecam security player <n>              security cam, track player
  /dronecam security pos                     security cam, track look-at position
Free-fly: WASD move  QE up/down  Shift fast  RMB/arrows rotate  F8 toggle";
```
